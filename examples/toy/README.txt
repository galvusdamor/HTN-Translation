Here are a couple of toy examples and basic instructions on how to run them with FF-X (Fast-Forward with axioms)

htntranslate -i 4 basic.hpddl basic-problem.hpddl &&  ff -o basic.pddl -f basic-problem.pddl

htntranslate -i 5 ordering.hpddl ordering-problem.hpddl &&  ff -o ordering.pddl -f ordering-problem.pddl

htntranslate -i 4 ordering2.hpddl ordering-problem.hpddl &&  ff -o ordering2.pddl -f ordering-problem.pddl
